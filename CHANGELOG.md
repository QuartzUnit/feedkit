# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-17

### Added
- Initial release
- 444 curated, verified RSS/Atom feeds across 6 categories (academia, finance, pets, science, society, technology)
- Async parallel feed collection with configurable concurrency
- SQLite storage with FTS5 full-text search
- MCP server with 9 tools for Claude Code / MCP clients
- OPML import and export
- Catalog search by keyword, category, and language
- Subscription management (subscribe, unsubscribe, list)
- CLI interface with 10+ commands
- Python async API
- Multi-language support (en, ko, ja, zh)
- 34 tests
